## 🧠 What is this?

This project is an **interactive developer portfolio built like a LEGO world**.

Instead of scrolling through static pages, you:

🧱 Explore
🎮 Interact
🔍 Discover

Every section is designed as a **building block**, snapping together to create a complete experience.

---

## ✨ Core Experience

🧱 **Modular LEGO UI**
A design system where everything feels like connected blocks

🎮 **Interactive Exploration**
Not just click → *play with the interface*

🏆 **Golden Brick Hunt**
Hidden elements across the site → find them all

🧍‍♂️ **Mini-Figure System**
Switch between different character styles

🏗️ **Scroll-Based Experience Tower**
Content builds as you scroll

🎮 **3D Project Viewer**
Rotate, zoom, and interact with projects

🤖 **AI Assistant (BrickBot)**
Ask questions → get guided through the portfolio
